# 工作计时器

一个悬浮在桌面的轻量级工作计时工具，专为不想看表、不想被弹窗打扰的用户设计。

## 功能特点

- 🪟 **半透明悬浮窗**：始终置顶，不遮挡视线，透明度可调
- ⏱️ **正计时/暂停**：随时暂停去上厕所，回来继续
- 📋 **任务管理**：点击按钮弹出窗口，选择或新建任务名
- 💾 **一键导出Excel**：自动记录任务名称、起止时间、总时长
- 🎨 **纯色视觉反馈**：结束或保存时仅时间标签闪烁提示，**无弹窗、无声音**
- 🔒 **数据本地存储**：任务列表保存在 `tasks.json`，不会上传到任何服务器

## 使用方法

### 直接运行（需要Python环境）

```bash
pip install PyQt5 openpyxl
python 计时器.py
```

### 打包成exe（无需Python环境）

```bash
pip install pyinstaller
pyinstaller -F -w --name "工作计时器" 计时器.py
```

打包后在 `dist` 文件夹找到 `工作计时器.exe`。

## 操作说明

| 操作 | 说明 |
|------|------|
| 点击“选择任务” | 弹出窗口，选择已有任务或新建任务 |
| 点击 ▶ | 开始计时（按钮变橙色） |
| 点击 ⏸ | 暂停计时（按钮变绿色） |
| 点击 ⏹ | 结束当前任务，自动记录（时间标签闪“✔ 已记录”） |
| 点击 💾 | 导出Excel文件（时间标签闪“✔ 已保存”） |
| 右键窗口 | 调整透明度、切换置顶 |

## 右键菜单

- 滑动条：调整窗口透明度（10%~100%）
- 切换置顶：让窗口始终保持在最上层

## 技术栈

- Python 3.12
- PyQt5（界面）
- openpyxl（导出Excel）
- PyInstaller（打包exe）
